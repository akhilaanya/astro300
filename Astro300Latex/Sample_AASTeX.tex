\section{Using References}

If you want to use a reference from your \texttt{MyRefs.bib} file you have to use the \texttt{\textbackslash cite} command.

\vspace{1em}

The \texttt{\textbackslash citet\{KEY\}} command will place parentheses around the year only. This is useful for references used as part of a sentence:

\vspace{1em}

Most of the words in this document come from \citet{1983Sci...220.1339R}.

\vspace{1em}

The \texttt{\textbackslash citep\{KEY\}} command will place parentheses around whole reference. This is useful for references that refer to the sentence but is not part of the sentence:

\vspace{1em}

``There is an art to flying, or rather a knack. The knack lies in learning how to throw yourself at the ground and miss. ... Clearly, it is this second part, the missing, that presents the difficulties.''
 \citep{adams1979hitchhiker}.
 
\vspace{1cm}
