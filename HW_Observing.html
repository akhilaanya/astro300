Determine the length of the observing run at each location

  • Length of run is defined by Astronomical Twilight
  • If your length is negative (or > 24 hr), you have to use a different which = parameter above
  • Your output should look like - At LOCATION the total observing time is TIME hours
  • The observing length should be VERY different at each location
In[49]:
# Location 1
chile_observing = chile_location.tonight(chile_noon, horizon = -18 * u.deg)
chile_observing_length = (chile_observing[1] - chile_observing[0]).to(u.h)
